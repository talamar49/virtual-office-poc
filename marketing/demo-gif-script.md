# 🎬 Demo GIF / Video Script — Virtual Office

## גרסה A: GIF קצר (15 שניות) — לסושיאל מדיה

### מה מצלמים
screen recording של Virtual Office בזמן אמת, ב-loop חלק.

### תסריט פריים-פריים

**[0-2 שניות]** — Opening shot
- המשרד האיזומטרי נטען — agents מופיעים בזה אחר זה
- Dark background, pixel font title

**[2-5 שניות]** — Agents זזים
- 3-4 agents עוברים מ-lounge לwork zone (status שינוי)
- Task bubbles מופיעות מעל ראשיהם
- Status dots מאירים בירוק/כחול

**[5-9 שניות]** — Interaction
- לחיצה על סוכן → detail panel נפתח
- הקלדת הודעה → שליחה
- Chat bubble מופיעה מעל הסוכן בcampus

**[9-12 שניות]** — Dashboard
- מעבר ל-Dashboard mode (📊)
- Status cards של כל הצוות
- Timeline מראה שינויי סטטוס

**[12-15 שניות]** — Logo + CTA
- fade in: *"Virtual Office by OpenClaw"*
- *"See your AI team. Live."*
- URL האתר

### הוראות הפקה (לנועה/עומר)
- רזולוציה: 1200×800 (16:10)
- Frame rate: 30fps → GIF: 15fps
- כלי: LICEcap / ScreenToGif / Kap (Mac)
- גודל יעד GIF: מתחת ל-5MB לסושיאל
