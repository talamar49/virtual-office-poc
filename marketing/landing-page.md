## SOLUTION SECTION

### כותרת
**"Virtual Office — המשרד שהצוות שלך ראוי לו"**

### 3 Feature Cards

**1. 👁️ ראה הכל בזמן אמת**
כל סוכן מופיע כדמות pixel art במשרד איזומטרי. ירוק = עובד. כחול = בתהליך. אפור = אופליין. תמיד יודע מה קורה.

**2. 💬 תקשר ישירות**
לחץ על סוכן, שלח הודעה, קבל תגובה — הכל מממשק אחד. לא צריך לפתוח terminal.

**3. 🏢 צוות שמרגיש חי**
עם emojis, שמות, ותפקידים — הסוכנים שלך מקבלים identity. זה לא רק כלי, זה **צוות**.

## Design Specs

### צבעים
- Background: `#1a1a2e` (dark navy)
- Primary: `#4a6aff` (pixel blue)
- Accent: `#e0e0e0` (light text)
- Success: `#4CAF50` / Working: `#2196F3` / Error: `#f44336`

### פונטים
- Headers: `"Press Start 2P"` (pixel font — consistent עם המוצר)
- Body: `"Segoe UI"` / system-ui (קריא)

### סגנון כללי
- Pixel art aesthetic — borders squared (no border-radius)
- Inset box shadows: `inset -2px -2px 0 #0a0a1a, inset 2px 2px 0 #2a2a4a`
- Dark theme throughout
- CTAs: pixel-style buttons עם hover states

### Layout
- Mobile-first
- Hero: full-width, GIF/animation centered
- Features: 3-column grid (desktop) / single column (mobile)
- Max-width: 1100px, centered
